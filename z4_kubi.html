Teoria – Zadanie 1

  1. Kiedy powstał JavaScript (podaj datę): 1995 r.
  2. Dlaczego jest to język skryptowy: Ponieważ jest interpretowany i nie wymaga kompilacji.
  3. Gdzie jest wykonywany JS (client lub server): Może działać zarówno po stronie klienta (przeglądarka), jak i serwera (np. Node.js).
  4. Dlaczego JS to bezpieczne narzędzie: Kod działa w środowisku przeglądarki i nie ma bezpośredniego dostępu do zasobów komputera użytkownika.
  5. Czy jest to język obiektowy, jeśli tak – to dlaczego: Tak, ponieważ wykorzystuje obiekty i umożliwia programowanie obiektowe.
  6. Wymień zalety JS:
    • Łatwa integracja z HTML i CSS
    • Wysoka wydajność w przeglądarkach
    • Ogromna społeczność i bogata biblioteka narzędzi
  7. Frameworki JS: React, Angular, Vue.js
  8. Dla aplikacji mobilnych można użyć frameworku: React Native
  9. AJAX to: Asynchronous JavaScript and XML – umożliwia komunikację z serwerem bez przeładowania strony
